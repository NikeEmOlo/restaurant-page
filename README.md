# Ramen Restaurant Page 🍜

A dynamic single-page restaurant website built with vanilla JavaScript and Webpack.
Built as part of The Odin Project JavaScript course.

## 📋 About
A fully JS-rendered restaurant website featuring a home hero page, 
dynamic menu, and contact page. All content is rendered on demand 
using ES6 modules — no HTML content, just JavaScript.

## ⚙️ Built With
- Vanilla JavaScript (ES6+)
- Webpack
- CSS3

## ✅ Features
- Tab-based navigation with active state tracking
- Dynamic menu rendering from JSON data using classes
- Contact form built entirely with JavaScript
- ES6 modules with named and default exports
- Object and class-based DOM rendering

## 📁 Project Structure
src/

├── menu.js         # Menu page module
├── contact.js      # Contact page module
